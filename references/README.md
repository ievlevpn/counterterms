# Reference

This project implements the renormalization combinatorics of

> I. Bailleul & M. Hoshino, *A tourist's guide to regularity structures and singular
> stochastic PDEs*, arXiv:2006.03524 (v3).

The paper is **not redistributed here** (it is the authors' copyright). Get it from arXiv:

- https://arxiv.org/abs/2006.03524

Throughout the code and notes we cite the paper by **`.tex` line number** (e.g. "tex 4003"),
referring to the v3 LaTeX source on arXiv. Place a local copy at
`references/tourist_guide.{pdf,tex}` (gitignored) to follow those citations.
